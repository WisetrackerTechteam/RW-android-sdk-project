3. SDK .JAR 파일 추출

    **유니티 프로젝트에 포함될 SDK .jar 파일 추출시 적용**

    - BASE 모듈 jar 추출
    안드로이드 스튜디오 Gradle/base/Tasks/other/exportjar 클릭
    -> 프로젝트경로/base/release/wisetracker-base.jar 생성

    - DOT 모듈 jar 추출
    안드로이드 스튜디오 Gradle/dot/Tasks/other/exportjar 클릭
    -> 프로젝트경로/new_dot/release/wisetracker-dot.jar 생성

    - DOX 모듈 jar 추출
    안드로이드 스튜디오 Gradle/dox/Tasks/other/exportjar 클릭
    -> 프로젝트경로/dox/release/wisetracker-dox.jar 생성
